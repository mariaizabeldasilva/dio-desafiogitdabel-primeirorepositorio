Desafio das Tags HTML

Você sabe o que são tags? Para que elas servem? Neste desafio, irei apresentar o conceito de tags e quais funções elas exercem na criação de um website

O que são tags?Voltar

É uma forma de organizar e classificar informações. Tag em inglês quer dizer etiqueta, na internet elas servem justamente para ajudar na organização das informações, agrupando aquelas que receberam a mesma marcação, facilitando encontrar outras relacionadas. São comandos na qual denominamos como TAGS

Para que servem?Voltar

Uma TAG serve para informar tamanho, dar descrição de algum conteúdo e promover mudanças através de seus atributos que dará resultados profissionais a sua página, de acordo com os valores predeterminados pelo desenvolvedor

Tags usadas neste desafioVoltar

Neste desafio usamos as TAGs de cabeçalho, que são:

  • h1 até h6: elas indicam os níveis de relevância do texto, sendo a h1 mais relevante e a h6 menos relevante
  • a TAG p: que significa ou que se tem a intenção de colocar um parágrafo

Também usamos as TAGs de formatação de texto para que a página visitada seja também aceita pelo usuário é importante que tenha uma boa apresentação do seu conteúdo

Algumas TAGs de formatação de texto:

  • TAG i: para dar ênfase no texto Deixando o texto desta forma
  • TAG strong: dá maior destaque a determinado trecho do texto
  • TAG abbr: indica uma abreviação que pode ser útil aos navegadores e sistemas ortográficos, como por exemplo:

    CBF foi fundada em 20 de agosto de 1914.
